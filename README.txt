Avant d'exécuter le programme :
-> Vérifier le chemin d'enregistrement des fichiers (methodes "fiche_club()" de la classe Club
et "tableau_score()" de la classe Championnat).
-> Vérifier l'emplacement de création de la BDD (Creation_BDD.py).
-> Vérifier que le fichier BDD_joueurs.db dans le dossier BDD n'existe pas encore.

Pour exécuter le programme :
-> Lancer le fichier fichier_test.py.

Pour exécuter les tests unitaires :
-> Lancer le fichier TESTS.py
